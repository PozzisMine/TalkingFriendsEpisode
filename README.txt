Сайт для просмотра эпизодов (статический, тёмная тема)

Как использовать:
1. Распакуй архив и загрузите файлы в репозиторий на GitHub.
2. Включи GitHub Pages (ветка main, папка root).
3. Открой index.html через GitHub Pages.

Редактирование:
- Данные находятся в data/shows.json.
- Для каждого сериала заполните: id, name, poster_url, description, episodes (массив).
- Для эпизодов: id, number, title, date, description, video_url. Для YouTube используйте embed-ссылку:
  https://www.youtube.com/embed/VIDEO_ID
